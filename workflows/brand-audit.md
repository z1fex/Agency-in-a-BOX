# Brand Audit
> **Teams:** Strategy, Marketing (Brand), Content | **Estimated steps:** 6

## What You'll Get
- Current brand analysis with strengths and inconsistencies
- Competitor brand landscape comparison
- Voice audit with channel-by-channel scoring
- Messaging framework (positioning, value props, elevator pitches)
- Brand guidelines (mission, vision, values, voice, personality)
- Editorial style guide with do/don't examples

## Prerequisites
- Active client profile in vault (run `onboard` first)

## Steps

### Step 1: Current Brand Analysis
**Read:** `agents/marketing/brand/voice-manager.md`
**Do:** Audit the client's current brand presence — website messaging, tone of existing content, social media voice, customer-facing materials. Document inconsistencies and strengths. Use web fetch to analyze the client's site.
**Save:** `vault/03-Research/[client]/brand-audit-findings-[date].md`

### Step 2: Competitor Brands
**Read:** `agents/intelligence/competitor-monitor.md`
**Do:** Analyze competitor brand positioning — voice, messaging hierarchy, content themes, brand personality. Identify category conventions and differentiation opportunities.
**Save:** `vault/04-Intelligence/[client]/competitor-brands-[date].md`

### Step 3: Voice Audit
**Read:** `agents/marketing/brand/voice-manager.md`
**Do:** Define current voice vs. aspirational voice. Score consistency across channels on dimensions: formal/casual, serious/playful, technical/accessible, authoritative/approachable. Identify gaps between current and desired voice.
**Save:** `vault/03-Research/[client]/voice-audit-[date].md`

### Step 4: Messaging Framework
**Read:** `agents/strategy/growth-strategist.md`
**Do:** Build messaging framework — brand promise, positioning statement, 3-5 value propositions, elevator pitches (30s, 60s, 2min), key messages per audience segment, and proof points for each claim.
**Save:** `output/[client]/[date]/brand-audit/messaging-framework.md`

### Step 5: Brand Guidelines
**Read:** `agents/marketing/brand/voice-manager.md`
**Do:** Create brand guidelines — mission, vision, values, personality traits, voice characteristics with do/don't examples, messaging hierarchy, boilerplate copy, and tagline options.
**Save:** `output/[client]/[date]/brand-audit/brand-guidelines.md`

### Step 6: Style Guide
**Read:** `agents/content/copywriter.md`
**Do:** Create editorial style guide — grammar preferences, capitalization rules, terminology glossary (approved/banned terms), formatting standards, channel-specific guidelines (email, social, blog, ads), and example sentences showing voice in action.
**Save:** `output/[client]/[date]/brand-audit/style-guide.md`

### Final Delivery
1. Run quality gate (`quality/qa-checklist.md`) on all deliverables
2. Save all deliverables to `output/[client]/[date]/brand-audit/`
3. Update `vault/00-Dashboard/Agency Dashboard.md` with brand audit summary
4. Update `vault/00-Dashboard/ROI Tracker.md` — log brand audit, messaging framework, brand guidelines, style guide
